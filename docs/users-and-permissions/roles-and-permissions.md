# User Roles & Permissions

## 1. Customer (Buyer)

**Access Level:** Basic authenticated user

**Capabilities:**

- Browse products (no auth required)
- Search and filter products
- View product details
- Add products to cart
- Manage shopping cart
- Place orders
- View order history
- Track order status
- Leave product reviews
- Manage profile
- Update shipping addresses
- View purchase history

---

## 2. Vendor (Seller)

**Access Level:** Authenticated vendor

**Capabilities:**

- All customer capabilities +
- Create/edit/delete own products
- Manage product inventory
- View sales analytics
- Process orders for their products
- Update order status
- View earnings/revenue
- Manage vendor profile
- Upload product images
- Set product pricing
- Create product variants (size, color)

---

## 3. Admin

**Access Level:** Full system access

**Capabilities:**

- All customer and vendor capabilities +
- Approve/reject vendor applications
- Manage all users (customers, vendors)
- View all orders across platform
- Manage all products
- View platform analytics
- Generate reports
- Manage categories
- Handle disputes
- System configuration
- Content moderation
